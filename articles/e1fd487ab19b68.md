Chrome の便利な拡張機能を厳選して紹介します。各項目は公式ページへのリンク、短い説明、主な用途と使い方のヒントを載せています。

## [React Developer Tools](https://chromewebstore.google.com/detail/react-developer-tools/fmkadmapgofadopljbjfkapdkoienihi?hl=ja&utm_source=ext_sidebar)

- 説明: React コンポーネントの階層や `props` / `state`、フックの状態をブラウザ上で確認・検査できる公式ツールです。
- 主な用途: コンポーネントのレンダリング挙動や状態管理のデバッグ、パフォーマンス調査。
- 使い方ヒント: DevTools の `Components` タブでコンポーネントを選択し、`Props` や `State` を確認。Hooks の状態も表示されるため、フック周りの不具合調査がしやすいです。

## [Obsidian Web Clipper](https://chromewebstore.google.com/detail/obsidian-web-clipper/cnjifjpddelmedmihgijeibhnjfabmlf?hl=ja&utm_source=ext_sidebar)

- 説明: ウェブページを Obsidian のノートとしてクリップ（Markdown 形式で保存）できる拡張機能です。
- 主な用途: 記事やレシピ、ドキュメントのクリップ保存、リサーチ資料の整理。
- 使い方ヒント: 拡張の設定で保存先の Vault とフォーマット（Markdown 変換オプション）を調整。クリップ時にメタデータやタグを付けると後から探しやすくなります。

## [URL抽出](https://chromewebstore.google.com/detail/url%E6%8A%BD%E5%87%BA/ggiihlkbikggfknjgbocmogobagckdpc?hl=ja&utm_source=ext_sidebar)

- 説明: 表示中のページからリンク（URL）を一括で抽出してコピーできるシンプルなツールです。
- 主な用途: ページ内の外部リンクや参考リンクをまとめて収集する際に便利。
- 使い方ヒント: アイコンをクリックして抽出結果を確認・フィルタリングし、クリップボードへコピー。必要に応じて正規表現やドメイン除外ルールを使うと使い勝手が上がります。

## まとめ

日常的なデバッグ、知見の蓄積、リンク収集に役立つ3つをピックアップしました。用途に合わせて導入してみてください。ほかにも気になる拡張があれば追記します。
